04.open属性

05.如何多个实现
多个超类具有相同的方法,  子类为了表示采用从哪个超类型继承的实现的方法，使用由尖括号中超类型名限定的 super 来指定那个超类的方法

class C4 : C() , A{
    override fun eat() {

    }

    override fun interA() {
        //调用A接口类interA方法
        super<A>.interA()
        //调用C类interA方法
        super<C>.interA()
    }
}
